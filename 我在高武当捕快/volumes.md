# volumes.md

## 总体结构
- 总字数：240万字
- 预计章节：800章
- 单章字数：3000字左右
- 分卷数：12卷
- 分卷原则：每卷解决一个阶段性问题，同时暴露更深一层规则；每卷结尾必须回收一部分伏笔，并把压力推向下一层地图、势力或世界规则。
- 长篇节奏：前期县域破案和系统规则，中期郡州、六扇门、世家宗门、朝堂博弈，后期禁地、古神、天刑印和王朝原罪。

## 卷一：库银无声
- 卷名：库银无声
- 预计章节范围：第1-66章
- 预计字数：约19.8万字
- 显性目标：方朔在清河县衙库银失窃案中自保，查清失窃手法，建立“读心术只能给线索、证据链才能结案”的基本规则。
- 隐藏危机：失窃库银中混有不可查名目的镇邪银，县衙有人试图让方朔做替罪羊。
- 阶段反派或阻力：王砚之、黑风帮转运人、乡绅钱庄、县衙内鬼体系。
- 主角状态变化：练骨小捕快 -> 练髓边缘；从只想躺平保命，到愿意主动补完整证据链。
- 关键角色关系变化：方朔与沈知微从互相提防到临时搭档；许敬安从压制方朔到承认其查案价值；楚照影以黑风帮灰色人物身份登场。
- 主要地点或舞台：清河县衙、库房、账房、赌坊、码头、黑风帮据点、义庄。
- 埋设伏笔：镇邪银名目、王砚之双账、沈知微暗查身份、许敬安旧案残卷、系统吞银异常。
- 强化伏笔：读心术读出偷鸡摸狗等小恶，提示地方秩序腐烂；库房封条完好，暗示内外勾连。
- 回收伏笔：回收库银失窃手法、县衙内鬼、黑风帮转运线路；确认系统破案奖励必须依赖真凶和证据链。
- 卷结局：王砚之暴露并留下半份暗账，黑风帮断尾，方朔获得奖励和小幅升职，但上游真正买家逃脱。
- 下一卷压力：暗账指向药铺、私盐和码头黑市，清河县的小案开始牵出更大的银钱网络。
- 当前进度：第1-35章已完成。第21-25章完成“第二本账边角”小闭环：旧渠车痕和西夹十五轴孔残码取得“乙九 / 西十五 / 旧料”，冯书吏认错被拆成替罪入口，何七家中灶灰与“别留整页”证词证明暗账被拆，西夹十五被确认为物的路线和车脚项，旧料柜垫板拓出多组同宽细纸压痕，确认第二本账非完整得见但边角已现。第26-30章完成“反扑与周福线”小闭环：旧例反扑未能压回小错，周福纸边与旧盐税副册第六册缺口初步相合，永丰旧册承认非明册草记，南湾假线暴露柳桩渡/郡道驿方向，县令准拟有限上报郡府；但郡府邵书办在文书出城前已到清河并要求带走物证，县衙转入保护旧案物证和应对郡府压力。第31-35章完成“郡府调物与暗封转移”小闭环：县衙分层转移旧案物证并发现封签异常，邵书办照验抄件并追底稿，王砚之誊清稿改写被识破，柳桩渡/郡道驿方向提供周福或账边被转移的外部证据，县令最终批定“旁验抄件，不得移原”。下一步进入第36-40章，应围绕周福可能尚在、郡府正式公文、柳桩渡驿站封牌痕和王砚之更大替罪链推进。

## 卷二：假药私盐
- 卷名：假药私盐
- 预计章节范围：第67-133章
- 预计字数：约20.1万字
- 显性目标：方朔追查假药害命案和私盐走私案，打通黑市、药铺、码头三条线。
- 隐藏危机：假药不是普通牟利，而是用被污染的药渣稀释武者丹药；私盐路线同时在转运镇邪银残料。
- 阶段反派或阻力：药铺掌柜、私盐牙人、黑风帮旧派、保护伞县丞。
- 主角状态变化：练髓 -> 九品入门；升级方式从单纯破案奖励转为“脏银归档后充值 + 武道感悟沉淀”。
- 关键角色关系变化：方朔与楚照影从敌对转为互相利用；沈知微开始怀疑方朔截留脏银的动机；许敬安给出郡府旧案线索。
- 主要地点或舞台：药铺、码头、盐仓、黑市、贫民巷、清河县外水道。
- 埋设伏笔：污染药渣来源、盐道背后的郡府批文、楚照影早年转运过镇邪银。
- 强化伏笔：镇邪银不只是银钱，而会影响人和尸体；系统对涉案脏银反馈更强。
- 回收伏笔：回收黑风帮外围转运线、假药真凶、县丞保护伞；让楚照影掌握的黑市账本进入主线。
- 卷结局：方朔抓住假药主犯，保下贫民证人，正式踏入九品；但药渣源头被提前焚毁。
- 下一卷压力：焚毁药渣的人留下连环命案痕迹，清河县开始出现失踪人口。

## 卷三：雨夜连环
- 卷名：雨夜连环
- 预计章节范围：第134-200章
- 预计字数：约20.1万字
- 显性目标：破解雨夜连环命案与失踪案，证明所谓妖邪杀人背后有人为操控。
- 隐藏危机：凶手在用失踪者试验污染药性，准备制造第一处小型禁地裂口。
- 阶段反派或阻力：伪装成邪祟受害者的丹师、郡府捕吏、被买通的仵作助手、低阶邪祟污染。
- 主角状态变化：九品 -> 八品；升级方式转为“验尸、现场复盘、实战破局”带来的武道感悟，不靠大量充值。
- 关键角色关系变化：张仵作成为稳定技术盟友；沈知微公开部分六扇门身份；许敬安旧案首次与郡府关联。
- 主要地点或舞台：清河县雨巷、义庄、乱葬岗、废弃药窑、郡府驿站。
- 埋设伏笔：尸体污染不完全符合妖邪规律、郡府捕吏提前到场、禁地裂口的银灰色痕迹。
- 强化伏笔：强者和官府可以借妖邪名义掩盖人为罪案；读心术对污染目标出现噪声。
- 回收伏笔：回收假药源头、失踪人口去向、第一处污染试验点；证明“妖邪说”是遮掩人祸。
- 卷结局：小型裂口被封，丹师伏法，郡府捕吏逃回上级庇护；方朔获得六扇门外围关注。
- 下一卷压力：郡府发来调令，要求方朔协查乡绅盐账，实为试探他掌握了多少。

## 卷十二：天刑不灭
- 卷名：天刑不灭
- 预计章节范围：第734-800章
- 预计字数：约20.1万字
- 显性目标：方朔进入古神遗迹，公开天下罪证，完成对皇室、国师府、世家、宗门和自身妥协的最终审判。
- 隐藏危机：无罪天并非可被武力斩杀的实体，而是寄生在强者默认无罪、罪名无人承担的世界规则中。
- 阶段反派或阻力：无罪天、顾行舟终局外壳、皇室残余、拒绝公审的宗门世家、天刑印代价。
- 主角状态变化：大宗师 -> 陆地神仙级终局 -> 法相化入天刑印；升级方式不是继续变强，而是以修为、肉身和自由换取秩序补全。
- 关键角色关系变化：沈知微接过人间六扇门秩序；姬玄衡被审判后承担镇邪赎罪；顾行舟认罪并失败；楚照影从灰色线人转为新秩序的边缘监督者。
- 主要地点或舞台：古神遗迹、坍塌天刑司、皇城审坛、天下卷宗法场、新六扇门总衙。
- 埋设伏笔：新捕快拾起黑金小令，方朔残响仍在天刑体系中。
- 强化伏笔：天刑印只承认证据、公议和执法代价；个人无敌不能替代制度落地。
- 回收伏笔：回收系统真相、无罪天本质、开国契约、镇邪银、顾行舟执印失败、方朔终局代价。

- 下一卷压力：全书完结余波。新秩序仍不完美，但罪必须被追问，天下捕快从此有了抬头可见的规矩。
