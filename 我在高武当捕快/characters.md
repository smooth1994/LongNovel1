## 主角
### 方朔
- 性别/年龄/身份：男，24岁，清河县底层捕快，现代打工人穿越者
- 外貌或标志性特征：身形不算魁梧，眼神总像在观察别人脸色；腰间挂旧捕刀
- 背景：前世做过运营/行政，擅长察言观色、背锅甩锅、流程留痕，爱看悬疑推理小说。穿越后继承练骨境小捕快身份，开局卷入库银失窃案。
- 表面追求：保命、赚钱、升职、别背锅。
- 深层渴望：在强者横行的世界里，找到属于小人物的安全感和掌控感。
- 灵魂需求：从“只想自保”成长为“愿意为公理承担代价”的执法者。
- 暗藏秘密或弱点：系统来源不明；爱财怕死容易被误解，也确实可能被利益诱惑。
- 初始状态 -> 触发事件 -> 认知失调 -> 蜕变节点 -> 最终状态：怂中带滑的底层捕快 -> 卷入库银失窃案并激活系统 -> 发现读心术不能替代证据，强者也能逃避律法 -> 证人被灭口、冤案滋生邪祟后开始主动追案 -> 以自身法相补全天刑印，成为镇压“强者无罪”的秩序核心
- 关系/价值冲突：与沈知微互补又冲突；与顾行舟在“真相是否必须公开”上对立
- 合作纽带：与沈知微、许敬安共同破案，建立六扇门证据网络
- 隐藏背叛/反转可能性：为破案可能短暂偏向灰色资源，但最终仍会回到证据与公审
- 当前状态更新：第35章中，方朔已从追线索转入保护证据链和文书定义权。他参与暗封夜移，识破邵书办问话顺序、主簿房誊清稿改写和柳桩渡合法转手路线，明确认识到“谁先拿走原物、谁先写死周福”本身就是案情；当前坚持原物不离县衙，并警惕王砚之借周福证据目录铺设更大替罪链。

## 关键盟友

- 外貌或标志性特征：眉眼清冷，衣着利落，佩短刀与细链锁

- 初始状态 -> 触发事件 -> 认知失调 -> 蜕变节点 -> 最终状态：冷静守规矩的年轻捕快 -> 与方朔共同调查库银案 -> 发现合法流程也可能被权贵操控 -> 选择保护证据而非服从上令 -> 成为新六扇门核心执掌者

- 合作纽带：共享案件线索、六扇门资源和镇邪银调查权限
- 隐藏背叛/反转可能性：可能表面配合高层，暗中救方朔；也可能因家族旧案与方朔决裂

- 暗藏秘密或弱点：曾为了保住下属，放走过一个关键证人，间接导致后续灭门案

- 隐藏背叛/反转可能性：前期可能为保县衙稳定而隐瞒关键线索
