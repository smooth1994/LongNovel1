# 第025章 第二本账的边角

永丰旧册没有送来。

送来的是王砚之。

午后，县衙偏厅的门开着，风把桌上的封条吹得轻轻响。方朔、沈知微、许敬安正把西夹十五的车身泥样、脚行小折、轴孔残码、冯书吏口供和何七家灶灰证物并排摆开。

王砚之就站在门外。

他没有贸然进来。

先拱手。

“许捕头。”

许敬安抬眼：“王主簿来得巧。”

王砚之道：“听闻永丰旧册迟迟未至，县尊让我来问问案情进展。账房封柜已有一日，再拖下去，县衙公银出入也会受影响。”

方朔看着他。

王砚之没有提冯书吏。

没有提何七。

没有提西夹十五。

他一开口就是县衙运转。

很王砚之。

许敬安道：“坐。”

王砚之坐下，目光从桌上一件件证物掠过。

看到西夹十五横木压痕拓片时，他眼神停了一瞬。

很短。

可方朔看见了。

王砚之知道这种细纸压痕意味着什么。

王砚之很快收回目光：“这些，似乎都只是间接之物。”

沈知微道：“所以在复核。”

“复核可以。”王砚之温声道，“但不可把所有间接之物强行串成暗账。脚行有脚行私记，钱庄有废票边纸，主簿房有旧料便格，车轴里有纸，也可能是车夫私藏账条。”

方朔道：“王主簿解释得真全。”

“做账的人，总要先分门别类。”

王砚之看向桌上的轴孔残码拓本。

乙九。

西十五。

旧料。

他指尖轻轻在桌边点了一下。

“这三列残字，若放在脚行里，可以是柜号、车号、货名。若放在主簿房，也可能是旧料出入便格。若放在永丰，则可能是垫料登记。方捕快，它能说明有记录，却不能说明有第二本账。”

这句话说得很准。

准到方朔没法立刻反驳。

单独看，这三列确实可以被解释成许多东西。

问题是，所有能解释它的地方，都在案子里。

永丰。

脚行。

主簿房。

旧料。

每一个解释都在把自己绕进去。

可要让县令、堂审、卷宗承认，仍差一块不能被普通口径吞掉的东西。

许敬安没有急。

他看向沈知微：“何七家灶灰复验了吗？”

“张仵作刚送来初验。”沈知微取出一张验记，“灶灰中三片纸角，一片含旧票纸浆，一片有红蜡封痕，一片残留墨痕似‘料’字右旁。旧衣领口涩味与梁二颈痕、旧渠湿布气味相近，但只能记相近，不定药名。”

王砚之道：“何七私逃前烧毁废票边料，也会如此。”

沈知微点头：“可记为一种解释。”

王砚之看向她：“沈捕快倒公允。”

沈知微没理他。

陈六从门外进来，脸上带着急色。

“总捕头，冯书吏要补口供。”

王砚之抬眼。

许敬安道：“带进来。”

冯书吏被带进偏厅时，人比昨夜更憔悴。

他没有看王砚之。

从进门起，他眼睛就盯着地。

许敬安道：“说。”

冯书吏跪下。

“小的昨夜想了一夜。何七来主簿房那日，除了细纸，还看过旧料柜底的垫板。”

方朔心里一动。

“垫板？”

“旧料出入柜下面有一块薄木垫板，防潮用。小的以前抄便格，有时会把细纸压在垫板上拓线，免得墨透到底册。何七那日问照式，小的拿垫板给他看过。”

王砚之终于开口：“冯书吏，旧料柜已封，垫板并未列为账册。”

冯书吏身子一抖。

“小的知道。可垫板上……可能有压痕。”

方朔看向许敬安。

许敬安起身：“去账房。”

王砚之也站起：“我同去。”

许敬安看他：“自然。”

账房门上的封条仍在。

沈知微先验封，确认没有被动过，才开柜。

旧料出入柜底那道册状空压痕仍在。

冯书吏跪在门口，指着柜底最里侧。

“垫板在下面。”

两个老捕快将柜中旧料册、废纸、空白夹板逐件移出，沈知微逐项记录，最后才露出一块薄木板。

木板颜色暗，边角有潮痕。

看起来很普通。

普通到若不是冯书吏提起，没人会在意。

沈知微没有让人直接拿。

她先用白纸覆在板面，轻轻压了一下。

板面并不平。

有几道细细的凹痕。

像有人长期在上面垫着窄纸写字。

张仵作不在，拓痕只能先做初拓。

沈知微取出炭粉，用极轻的手势扫过白纸。

细线一点点显出来。

先是一道竖线。

再是几处断开的横。

接着，三个残缺的字形从白纸上浮出来。

乙九。

西十五。

旧料。

偏厅里所有人的呼吸都轻了一下。

这还不够。

第21章轴孔细纸也有这三个残码。

若只是重复，仍能说成同一类旧料便格。

沈知微没有停。

她换了一张纸，继续拓另一处压痕。

第二组更淡。

第一列残得厉害，只能看出一个“乙”字尾。

第二列是“西十五”。

第三列不是旧料。

是半个“灰”字。

灰。

沈知微的手停住。

方朔心口像被人捏了一下。

王砚之的脸色终于冷了。

冯书吏伏在地上，不敢抬头。

许敬安沉声：“继续。”

第三张拓纸上，只显出几笔。

第一列像“乙九”。

第二列有一个“西”。

第三列只剩“木”字旁。

木楔？

旧木？

不能定。

第四张拓纸上，出现了一个更短的残痕。

“乙九。”

“西十五。”

“旧布。”

这一次，字更清楚。

沈知微把几张拓纸按顺序摆开。

第一组：乙九 / 西十五 / 旧料。

第二组：乙? / 西十五 / 灰?

第三组：乙九 / 西? / 木?

第四组：乙九 / 西十五 / 旧布。

没有完整账页。

没有完整数字。

没有三千两。

没有王砚之名字。

可这些压痕不是车夫私记能解释的。

它们在主簿房旧料柜垫板上。

与车轴残码同宽、同列、同格式。

且不止一组。

方朔看着第二组那个半个“灰”字，忽然觉得旧渠的风又吹到了账房里。

冯书吏昨夜失口的“灰银”，何七家的“别留整页”，西夹十五横木细纸压痕，此刻都压到这块木板上。

王砚之先开口。

“旧料柜垫板上有旧料便格压痕，不奇怪。”

他的声音仍稳。

但没有刚才那么温。

方朔道：“一组不奇怪。四组也不奇怪？”

“旧料出入本就多次记录。”

“那为什么不在明册？”

“便格草录，未必入月账。”

“为什么有一组是灰？”

王砚之看向那张拓纸：“半字而已。灰布、灰麻、灰车轴，皆可。”

沈知微提笔，在拓纸旁写下：第三栏残字“灰”，具体料名待核。

王砚之看着她写。

方朔忽然明白。

王砚之不是不怕。

他怕的不是半个“灰”。

他怕的是这半个“灰”和其他东西并排。

单独的灰，可以是灰布。

单独的西十五，可以是车脚。

单独的乙九，可以是柜位。

单独的旧料，可以是垫料。

可四组压痕、车轴残码、何七细纸、脚行小折、永丰新册、冯书吏口供一起摆开，就不是旧例。

是对应。

方朔道：“王主簿，旧料便格不入月账，我认。”

王砚之看他。

“脚行车折会写车脚，我也认。”

“永丰垫料登记会写乙九，我也认。”

方朔拿起第21章车轴残码拓本，放到垫板第一组拓痕旁。

乙九 / 西十五 / 旧料。

两处几乎相同。

“可是同一组三栏，既在西夹十五车轴里，又在主簿房旧料柜垫板上，还由何七带细纸来问照式，再由脚行小折证明初七确有西夹十五到永丰乙九柜送垫料。”

他又放下第四组拓痕。

乙九 / 西十五 / 旧布。

“同一个柜位，同一个车脚，不止一次对应不同料名。”

最后，他指向第二组。

乙? / 西十五 / 灰?

“这已经不是车夫私记。它是一个格式。”

王砚之淡淡道：“格式不是罪。”

“对。”

方朔点头。

“所以我不说罪。我只说第二本账存在。”

这句话落下，账房里静得能听见封条边缘被风吹动的声音。

许敬安看着垫板拓痕。

他脸色很沉，却没有急着拍案。

“沈知微，写。”

沈知微提笔。

“旧料柜垫板见多组同宽细纸压痕，初拓可见乙九、西十五、旧料、旧布及残灰字样。其格式与西夹十五车轴残码相合，与冯书吏供述旧料三栏照式、何七携细纸来主簿房一节相互对应。此物可证明主簿房内曾存在不入明册的分散对照记录，非完整账册，待复拓、验墨、比对。”

每一句都克制。

每一句都留边界。

可每一句都像钉子。

钉在王砚之刚才那句“不能说明有第二本账”上。

王砚之看着沈知微写完，忽然笑了一下。

“沈捕快这手卷宗，真不像初到县衙。”

沈知微把笔放下。

“王主簿这手解释，也不像只管明账。”

王砚之笑意淡去。

方朔差点没忍住。

这姑娘平时不多话。

一开口，刀挺准。

许敬安将垫板重新封存。

“垫板入暗封。拓纸一份入明卷，一份入暗封，一份由我亲押。”

王砚之道：“为何入暗封？若只是旧料便格，无需暗封。若是案证，也该明示县尊。”

许敬安看他：“明卷写旧料柜垫板压痕。暗封留原拓，防止有人说拓纸被改。”

“我并非此意。”

“我也不是只防你。”

这话比指名道姓更重。

王砚之没有再争。

但他的目光落在那块垫板上，停了很久。

方朔知道，王砚之在算。

算这块垫板会把谁拖下水。

算还能不能把冯书吏推出去。

算韩贵够不够挡。

算永丰会不会割何七。

算县令愿不愿意为了半个“灰”字上报郡府。

这时，门外又有脚步声。

陈六急匆匆进来。

“总捕头，义庄那边来人。梁二醒了一次，说了半句话。”

方朔抬头。

“什么话？”

陈六看了一眼王砚之，有些犹豫。

许敬安道：“说。”

陈六低声道：“他说，周福没拿整账，只撕了边。”

账房里的空气像被这半句话压住。

周福没拿整账。

只撕了边。

这和何七家那句“别留整页”合上了。

完整账被人拿走。

边角被不同人撕开、带走、藏进车轴、压在垫板、烧在灶里。

第二本账不是没有。

是有人故意让它没有“整本”。

方朔看向王砚之。

王砚之脸上没有表情。

可他右手指尖的墨痕，第一次被他藏进了袖中。

许敬安缓缓道：“梁二这句话，暂不入明卷。义庄补录，待张仵作在场复问。”

沈知微点头。

方朔也没有异议。

梁二重伤，半句话不能直接定案。

但作为方向，它太重要。

周福没拿整账，只撕了边。

这解释了为什么他们一直找不到完整第二本账。

也解释了为什么旧渠、车轴、何七家、主簿房垫板都只剩边角。

有人拿走了整本。

有人撕下了边角。

有人想烧掉整页。

有人把边条藏进车轴。

每个人都只碰了一小段。

分罪。

分账。

分证。

方朔忽然觉得第7章沈知微说过的“分罪”二字，终于落到了账上。

王砚之站起身。

“既然梁二醒了，我不便打扰捕房办案。许捕头，旧料柜垫板之事，我会向县尊说明：此为旧料便格压痕，尚不足定罪。”

许敬安道：“你可以说明。”

方朔补了一句：“我们也会说明：尚不足定罪，但足以证明有不入明册的分散对照记录。”

王砚之看向他。

两人隔着柜门、封条和一块薄木垫板对视。

王砚之忽然道：“方捕快，你一直追账，有没有想过，账追到底，未必只有贼？”

方朔道：“还有什么？”

“还有规矩。”

王砚之声音很轻。

“有些账不入明册，不是因为有人贪，而是因为明册容不下。你若把所有暗处都翻出来，清河县未必承得住。”

这是警告。

也像劝。

方朔看着他。

“那就看清河县承不承得住罪。”

王砚之没有再说。

他转身离开账房。

衣袖经过门槛时，带起一点风。

封条轻轻晃了晃。

方朔低头看垫板拓纸。

乙九。

西十五。

旧料。

乙九。

西十五。

旧布。

乙。

西十五。

灰。

字都不全。

账也不全。

可正因为不全，才更像这桩案子的真相。

所有人都在拿不全当盾。

不全的票。

不全的册。

不全的口供。

不全的车痕。

不全的账码。

可方朔现在终于有了一块足以压住桌面的边角。

第二本账确实存在。

它不是完整摆在某个柜子里的册子。

它是藏在主簿房垫板、车轴细纸、何七灶灰、脚行小折和永丰新册空白之间的一套分散对照。

许敬安把垫板封好，递给老捕快。

“送暗库。”

老捕快应声离去。

沈知微收好拓纸，低声道：“第25章这个小闭环，可以写了。”

方朔看她。

她难得像开了个玩笑，可脸上没什么笑意。

方朔明白她的意思。

不是故事小闭环。

是案情摘要可以写一句新的话。

第20章，他们只能写：县衙里有第二本账。

现在，可以写得更准一点。

许敬安拿起笔，亲自在案情摘要上写下：

“查旧料柜垫板压痕、西夹十五车轴残码、冯书吏供述、何七住处焚纸痕及脚行小折，可相互印证永丰乙九、西夹十五、旧料/旧布/残灰字样存在分散对照。第二本账非完整得见，然其边角已现。”

他写完，停笔。

“下一步，查周福。”

方朔没有说话。

他知道这句话后面还有更多东西。

周福生死。

何七去向。

韩贵假线。

永丰旧册。

王砚之反扑。

县令是否上报郡府。

每一件都比前面更难。

可至少现在，他们不再只是说“怀疑”。

他们手里有了边角。

一块薄木垫板。

几张淡得快看不见的拓纸。

一条车轴里泡烂的细纸。

还有梁二醒来时挤出的半句话。

这些东西轻得很。

轻到风一吹就能散。

所以必须封。

必须写。

必须让它们进入卷宗。

方朔站在账房门口，看着日光落在封条上。

系统仍然没有奖励。

没有提示。

没有告诉他真凶是谁。

他反而觉得踏实了一点。

因为这一次，不是系统告诉他案子往哪走。

是证据自己把路压出来了。

旧料柜垫板上的压痕很浅。

可浅痕连起来，也能压住一座县衙。
