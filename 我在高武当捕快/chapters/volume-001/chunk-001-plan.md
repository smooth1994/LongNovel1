# 第一卷第1-5章 Chunk 写作计划

## 1. 第1-5章共同的小悬念目标

第1-5章要完成“库银失窃案开局小闭环”：方朔穿越后发现自己被推到库银案前线，为了不背锅被迫查案；系统与读心术登场，但明确读心术只能提供最近三天违法乱纪线索，不能直接定罪；库房封条完整、账册日期错位、衙门内部态度异常共同形成第一层谜面；第5章抓住一个表层小偷或替罪羊，证明方朔不是废物，也证明真正库银案远未结束。

## 2. 每章必须推进的情节事实

### 第1章 - 醒来先背锅
- 方朔醒来，确认自己穿越到大乾清河县衙底层捕快身上。
- 交代原主练骨境、无背景、低级捕快身份。
- 县衙库银失窃，县令限期破案，衙门上下急需替罪羊。
- 方朔发现自己因夜巡、身份低、好拿捏，被推到嫌疑边缘。
- 结尾必须让方朔意识到：不查案就可能被顶罪。

### 第2章 - 读心读出偷鸡贼
- 捕快奖励系统启动，开局给予读心术。
- 明确读心术规则：只能读取目标最近三天违法乱纪之事，每天10次。
- 用同僚偷鸡、赌债、藏私房钱等小恶制造轻松笑点。
- 方朔尝试读库银案相关人员，却发现读到的多是小罪，无法直接锁定真凶。
- 结尾必须形成认知：系统强，但破案仍要靠调查和证据链。

### 第3章 - 封条没破
- 方朔进入库房现场勘查。
- 明确库房封条完整、锁眼无撬痕、夜巡记录表面无误。
- 方朔发现账册日期、银锭批次或值夜记录存在细微不协调。
- 王砚之或账房体系首次进入嫌疑范围，但不能坐实。
- 结尾必须形成谜面：银子像是从“账”里消失，而不是从库房被偷走。

### 第4章 - 总捕头的眼神
- 许敬安正式压场，既敲打方朔，又暗中观察他的查案能力。
- 沈知微登场，表现出讲规矩、重流程、观察现场边界的特点。
- 方朔与沈知微产生第一次方法冲突：方朔想保命快查，沈知微要求程序和证据。
- 暗示沈知微不是普通新捕快，许敬安也知道旧案或更深内幕。
- 结尾必须让三人关系成形：方朔被迫查、沈知微盯规矩、许敬安压着局面。

### 第5章 - 第一只替罪羊
- 方朔根据读心术小线索、夜巡记录和库房外围痕迹，抓住一个表层偷银或偷物的小角色。
- 该小角色确实违法，能完成一个小案闭环。
- 证据证明他接触过库房外围或碎银，但没有能力完成核心库银失窃。
- 方朔避免第一轮背锅，获得衙门内有限认可或至少争取继续查案资格。
- 结尾必须留下更大疑点：真正库银案背后还有内鬼、账册或帮派中转线。

## 3. 每章必须变化的角色状态

### 第1章
- 方朔：迷茫穿越者 -> 被迫接受捕快身份和背锅危机。
- 许敬安：尚未完全信任方朔，只把他视为麻烦下属。
- 县衙群像：表现为人人想自保、没人愿担责。

### 第2章
- 方朔：恐慌求生 -> 发现系统可用，但不能无脑依赖。
- 系统：从未知外挂 -> 有明确限制的破案工具。
- 同僚群像：从普通背景板 -> 暴露小恶，形成轻喜剧质感。

### 第3章
- 方朔：被动应付 -> 主动观察现场并开始推理。
- 王砚之：从普通主簿/账房人物 -> 进入读者疑心范围。
- 案件状态：从“库银被偷” -> “库房未破但账册异常”。

### 第4章
- 方朔：单打独斗 -> 被纳入许敬安和沈知微的视线。
- 沈知微：陌生新捕快 -> 有能力、有规矩、有隐藏身份感的潜在搭档。
- 许敬安：压制上司 -> 可能藏着旧案经验的复杂上司。

### 第5章
- 方朔：可能背锅的人 -> 能抓住小线索的捕快。
- 沈知微：审视方朔 -> 初步承认方朔有查案价值。
- 许敬安：观察方朔 -> 给方朔继续查下去的有限空间。
- 表层嫌疑人：普通小偷 -> 被证据链锁住的小案真凶，但不是主案核心。

## 4. 每章要埋设或强化的伏笔

### 第1章
- 埋 F-001：库银失窃背后不是单纯盗窃。
- 埋 F-004：方朔即将获得受限读心术。
- 暗埋县衙内部有人急于找替罪羊。

### 第2章
- 强化 F-004：读心术只能读最近三天违法乱纪，且每日10次。
- 埋 F-005：系统奖励与银钱、破案有关，但规则不完全明朗。
- 埋轻喜剧模式：小恶可制造笑点，但不能替代主案推理。

### 第3章
- 强化 F-001：库房封条完整，失窃不像外盗。
- 埋 F-002：王主簿账册有双重记账痕迹。
- 暗埋“账册比库房更像案发现场”的后续认知。

### 第4章
- 埋 F-006：沈知微并非单纯来历练。
- 暗埋许敬安旧案残卷或旧伤。
- 强化“规矩与变通”的长期角色冲突。

### 第5章
- 强化 F-001：小偷无法解释核心库银失踪。
- 埋/强化 F-003：小角色背后可能连着帮派或外部中转线。
- 小回收：回收第1-4章外围痕迹和读心小线索，完成小案闭环。

## 5. 第5章的小闭环结果

- 方朔抓到第一只“替罪羊”级别的小偷或外围接触者。
- 该人确实犯案，证据链能成立，证明方朔的查案能力和系统线索价值。
- 但该人没有能力破解封条、改账或搬走核心库银。
- 方朔暂时摆脱被立刻顶罪的危机，获得继续调查的资格。
- 读者获得第一个小爽点，同时清楚知道主案更深：库银失窃的核心在账册、内鬼和外部中转。

## 6. 第6章需要承接的压力

- 小案闭环后，真正库银案仍未解决，县令限期压力继续存在。
- 王砚之和账房体系需要正式进入调查焦点。
- 被抓小偷可能供出碎银、赌坊、跑腿人或账房相关信息。
- 衙门内部有人开始意识到方朔不是好糊弄的替罪羊，后续会增加阻力。
- 沈知微会要求方朔按流程补证据，避免小案成果被上面压掉。

## 7. 这5章禁止重复的桥段

- 禁止连续用读心术直接解决问题；读心术每次只能给线索，必须落到证据。
- 禁止把所有衙役都写成看不起主角的脸谱角色。
- 禁止主角刚穿越就大范围装逼或嘴贱挑衅。
- 禁止第1-5章就揭露镇邪银、黑风帮、王砚之主谋等深层答案。
- 禁止每章都靠“某人心虚露馅”推进，必须混用现场、账册、口供、读心小线索。
- 禁止沈知微对主角无理由欣赏或倒贴，她只能初步承认其查案价值。
- 禁止许敬安直接变成主角靠山，他应保持压制、观察和有限放权。
- 禁止把小偷写成降智反派，他可以贪、小、怕，但要有合理动机和行动逻辑。
