# story-bible.md

## 故事核心种子

方长朔是赵国横断山脉外围的低资质散修，火、土、木、水四灵根，苦修十八载才到炼气五层。一次遗迹机缘中，他没有冒险，反而因邻居陆承渊重伤归来、临终托孤而得到一本以异界文字书写的《圣心诀》。他答应照顾道兄凡俗妻女，却从此走上“延寿、双修、灵植、丹药、家族经营”交织的长生之路。

整书不是单纯打怪升级，而是一个底层散修把“照顾妻女”发展成资源整合术，把“多女主关系”发展成修炼反馈系统，把“长生真气”发展成灵植、丹药、突破、治疗、傀儡和跨界生态的过程。

## 整书主线

托孤机缘 -> 圣心诀入门 -> 照顾妻女 -> 小家成型 -> 沈家入口 -> 罗家清算 -> 筑基成功 -> 丹药灵植产业 -> 后宅扩张 -> 金丹家族 -> 元婴宗门化 -> 皇朝与多国布局 -> 化神破界 -> 圣祖界与灵界 -> 嫂嫂回归 -> 小夭母亲 -> 仙府归家 -> 飞升仙界。

### 家族主线

### 感情与双修主线

## 主角初始设定

- 姓名：方长朔
- 身份：天元界赵国横断山脉外围散修
- 初始修为：炼气境五层
- 灵根：火、土、木、水四灵根，资质普通偏差
- 初始资源：无宗门、无家族、无强力靠山，只有散修经验和谨慎性格
- 初始目标：苟活、延寿、寻找筑基希望
- 初始矛盾：想做个有底线的人，但修仙界资源残酷，善心必须与手段绑定

## 主角性格锚点

- 谨慎：能不冒险就不冒险，遗迹、秘境、宗门冲突都先评估生还率。
- 会算账：人情、灵石、丹药、灵植、女修背景、子嗣灵根都能纳入收益计算。
- 表面温和：习惯以“照顾”“帮忙”“合作”包装利益扩张。
- 对自己人宽厚：愿意给资源、功法、丹药、突破护法和长期培养。
- 对敌人狠辣：能偷袭不硬打，能借刀不亲自动手，能变废为宝就做人傀。
- 灰色底色：有道德感，但会被长生、家族、情欲和资源逐渐侵蚀。

## 金手指总设定

### 名称

长生双修反哺系统，又可称为圣心长生系统。

### 表现形式

不是高频弹窗的明牌系统，而是由《圣心诀》、长生真气、双修反馈、熟练度成长、灵植催生、灵根同质秘法共同形成的隐性系统。写作时可以偶尔用“拟制面板”强化爽点，但正文中不宜每章机械弹窗。

### 底层闭环

双修/修炼/战斗/炼丹/催生灵植 -> 获得熟练度与生命反馈 -> 提升功法、术法、灵根、修为或资源产量 -> 壮大家族与后宫网络 -> 接触更高质量对象和资源 -> 继续反馈。

## 《圣心诀》规则

### 来源

来自陆承渊遗物，文字为方长朔前世文字。原为异界武道长生功法，在修仙世界被灵气强化。

### 核心作用

- 延寿
- 治疗与恢复
- 强化肉身
- 凝聚长生真气
- 辅助突破
- 催生灵植
- 精神攻击和控制
- 后期触及生命、死亡、法则和天地灵源

### 层级

- 第一层：延寿约二十载，初步强化恢复力。
- 第二层：延寿约五十载，恢复力增强。
- 第三层：延寿约一百二十载，凝聚长生真气，治疗、催生、辅助突破功能成型。
- 第四层：延寿约三百载，需要长生之物，解锁邪血劫、玄冰绝等手段。
- 第五层：延寿约八百载，需要强大生物眼睛辅助，解锁惊目劫、天宫幻影。
- 第六层：需要筑基境妖兽心脏等心类材料，解锁天心劫。
- 第七层：涉及元神层面，筑基阶段难以完全触及。
- 第八层：元婴前后修到高层，肉身、寿元、神识、遁法全面强化。
- 第九层：大圆满，可延寿万载，解锁七无绝境等强力手段。
- 第十层：后期由方长朔自行推演，以天地灵源、生命法则、青源圣母等高层资源开路。

## 长生真气规则

### 用途

- 治疗外伤、修补筋脉、接续肢体。
- 辅助别人突破，尤其是失败后的修补与保命。
- 催生低阶到高阶灵植。
- 治疗天地灵根，激活沉睡生命。
- 改造灵植生长环境，反哺地脉和家族修炼场。
- 配合人傀之术维持傀儡战力。

### 限制

- 早期只能小规模治疗和催生，不能无限复活。
- 大量输出会造成亏空，需要药草、妖兽气血、长生灵物、天地灵源或修行时间补回。
- 非方长朔修炼出的长生真气品质较低，效果约为主角的一部分。
- 对外必须隐藏真实机制，常用丹药、秘术、特殊体质、灵植机缘作为解释。

## 双修反哺规则

双修是修炼系统的重要分支，不是单纯情欲戏。每一次重要双修都应至少带来以下一种收益：

- 圣心诀运转熟练度提升。
- 术法熟练度提升。
- 灵元提炼经验增加。
- 特定属性灵气掌控增强。
- 对方修为、体质、灵根、秘法被主角吸收部分经验。
- 对方获得长生真气洗练、突破辅助或修炼速度提升。
- 双方关系绑定加深，背后家族、宗门、产业被纳入方家网络。

### 灵根同质

冰雪神殿秘法证明：与天灵根品质对象双修，可让灵根品质不足者被同质化提升。后期这成为方长朔补全五行、扩展风雷冰等异属性的重要路径。

## 熟练度系统规则

### 来源

- 双修
- 实战
- 炼丹失败和复盘
- 观摩突破
- 教导弟子
- 使用术法
- 催生灵植
- 阉割并传授圣心诀秘术

### 作用

- 提高施法和炼丹成功率。
- 减少消耗。
- 缩短修炼时间。
- 增强术法威力。
- 帮助主角拆分秘术，把惊目劫、天宫幻影等手段弱化后传授给核心成员。
- 让炼丹、御兽、灵植、阵法等副职业从偶然成功变成稳定产业。

## 写作禁区

- 不要把主角写成热血莽夫。
- 不要让双修失去修炼反馈。
- 不要让灵植、丹药、家族产业只出现一次。
- 不要让系统变成无代价万能许愿机。
- 不要让后宫人物只剩标签，必须有资源入口、情绪矛盾或长期功能。
- 不要提前泄露圣心诀全部能力。
- 不要在初始化阶段写章节正文。
