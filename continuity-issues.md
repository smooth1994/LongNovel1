# continuity-issues.md

## 可重复的桥段

这些桥段可以反复使用，但每次必须换资源、换关系压力、换收益层级，避免机械重复。

### 托孤照顾

- 基本结构：有人临终或落难 -> 把妻女/弟子/家族托付给方长朔 -> 方长朔以照顾为名接管关系和资源。
- 可变项：遗孀、女儿、徒弟、族人、宗门女修、妖族幼崽、小夭等。
- 注意：每次都要有道德压力和利益回报，不能只写成白捡后宫。
- 陆映棠是本书第一次托孤中的重要角色，后来改名为方映棠。前期必须有具体生活、安置、共处分寸、称呼、改名和归属感细节，不能被“家眷”二字概括掉。

### 陆映棠旧债转情缘

- 基本结构：失去依靠的成年女眷陆映棠警惕自保 -> 方长朔安置和保护 -> 逐渐接受方长朔 -> 进入方家事务/修炼体系 -> 改名方映棠 -> 成为方家早期情感锚点并在中期转入道侣线。
- 可变项：灵根检测、第一次修炼、第一次被敌人盯上、第一次主动维护方长朔、对许清蘅关系变化的反应。
- 注意：这条线前期以安置、试探、共处和家族归属为主，不要写成抚养关系，也不要一登场就直接进入双修推进。

### 女修绑定

- 基本结构：女修遇困境 -> 方长朔提供丹药、灵植、庇护、突破、治疗或资源 -> 关系软化 -> 双修/联姻/家族绑定 -> 女修背后资源进入方家。
- 可变项：沈家、林家、程家、宗门、妖族、冰雪神殿、灵界宗门。
- 注意：女修必须有独立功能，至少带来资源、技能、身份、情绪矛盾之一。

### 灵植催生

- 基本结构：不起眼灵植 -> 长生真气试探 -> 异常成长 -> 方长朔隐藏真实原因 -> 家族获得长期收益。
- 可变项：紫罗天香树、旋木灵藤、五色元灵树、墨玉柳、雪灵白莲、七叶剑草。
- 注意：长生真气有消耗，外界解释要合理，不能让所有人立刻知道真相。

### 熟练度成长

- 基本结构：第一次尝试失败 -> 复盘失败原因 -> 多人协作或双修补齐短板 -> 多次练习 -> 成功一次 -> 稳定成功 -> 成为产业。
- 可变项：炼丹、御兽、阵法、术法、灵植、精神秘术、人傀控制。
- 注意：失败次数和成本要写出来，成功才有爽感。

### 敌对家族清算

- 基本结构：敌对家族暴露恶行 -> 方长朔先收集情报 -> 借强者/规则/仇敌清算 -> 战利品转化为家族资源 -> 敌人妻女或旁支被吸纳。
- 可变项：罗家、林家、程家、宗门分支、妖族部族、灵界势力。
- 注意：主角不应无脑正面强攻，尤其低阶时期必须靠布局。

### 突破铺垫

- 基本结构：观摩别人突破 -> 记录失败经验 -> 准备丹药、灵脉、护法、长生真气 -> 低调闭关 -> 危机时靠恢复力和准备扛过 -> 突破后轻描淡写。
- 可变项：筑基、金丹、元婴、化神、天仙劫。
- 注意：越高境界越需要长期铺垫，不能临场开挂。

### 后宅吃醋与资源分配

- 基本结构：新女修进入 -> 老人不满或试探 -> 方长朔用资源、修炼安排、子嗣利益平衡 -> 新关系被纳入秩序。
- 可变项：沈婉宁、沈云棠、沈璇音、顾柔音、白灵绮、冷雪弦。
- 注意：后宅冲突要服务修炼和家族，不要变成纯吵架。

### 低调遮掩金手指

- 基本结构：方长朔制造奇迹 -> 外人震惊 -> 他用丹药、秘术、运气、灵植环境解释 -> 少数核心成员知道部分真相。
- 可变项：治疗、灵植催生、突破、炼丹成功、人傀修补。
- 注意：真相分层，不能让普通敌人轻易看穿圣心诀。

## 设定红线

- 方长朔不是正道圣人，也不是无脑恶人。他的核心是“有底线但利益最大化”。
- 方长朔不能主动去高危遗迹送死，除非有明确准备、收益和退路。
- 圣心诀不能开局全能力解锁，必须按层级逐步开放。
- 双修必须带来实际修炼、熟练度、灵根或关系收益。
- 后宫人物不能只做标签，每个核心女修必须有长期功能。
- 陆映棠/方映棠前期必须按重要角色处理，不能被省略成许清蘅的附属或一句“家眷”；改名节点必须有情绪与身份意义，道侣化转折也要有足够铺垫。
- 筑基丹、虚灵丹、化婴丹等关键丹药不能廉价化。
- 灵植成长可以被催生，但不能完全跳过资源消耗和时间成本。
- 低阶时期越级杀敌必须靠偷袭、阵法、傀儡、毒、情报或借刀。
- 人傀之术必须保持灰色和危险感，不能被写成毫无争议的常规善举。
- 方家扩张必须有管理、资源分配、后宅冲突和外部窥探，不可无成本壮大。
- 灵界阶段必须让主角重新感到压力，不能飞升后继续无敌平推。
- 终局必须落到“长生”和“回家”，不能只以战力第一收束。

## 重复风险清单

- 女修绑定模式过于相似：需要改变困境类型和收益类型。
- 炼丹失败复盘过多：重要丹药详写，普通丹药略写。
- 灵植催生重复震惊：每次换外部压力，例如抢夺、病变、渡劫、诞灵。
- 后宅吃醋过频：只在新关系改变资源格局时重点写。
- 家族交易太流水：每次交易要带出新势力、新风险或新伏笔。
- 主角过度解释金手指：用行动和结果表现，少让主角自白机制。
